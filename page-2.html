Tools

First, let's lay out all of the tools that we have from the very beginning.

  • Points
  • Grid

A point knows only the state it's in.

  • State - On/Off

The grid is a point container. Besides holding a bunch of points, it also has to have information about it's size. Without a specified size, there would be ambiguity as to the exact position of each dot, because a collection of n dots has n possible width and height combinations. You can only access points through the grid.

  • A collection of points
  • Horizontal size - width
  • Vertical size - height

Methods

First, we need to define a method of how to access specific points. In a 2-dimensional world you can define all possible points using a set of two basis vectors (one horizontal, one vertical):

  • h = (1, 0)
  • v = (0, 1)

To define a new position, we calculate the sum of the two basis vectors. The x and y factors indicate how far along the horizontal and vertical axis do we need to go to find the point.

position = (x * h) + (y * v) = (x, y)

To map between a given position (x, y) and the point number p, we need the following rule.

map(x, y) = p = y * height + x

That's all we need to start drawing.
