A point

This is a single point. By default, it's initial state is set to off, that's why it's a bit hard to see. You toggle between these two states using the following methods. Try to turn it on and see what it looks like then.

point.turnOn()
point.turnOff()

A grid

We collect points using a grid. Here's one below. It's width is 21 points and height is 21 points.

The grid has 2 dimensions, so to select a specific point, you have to provide how far along each dimension it is located. The horizontal dimension is organised left to right, the vertical dimension is top to bottom. Here are some locations for reference.

  • (0, 0) - top left
  • (20, 0) - top right
  • (20, 20) - bottom right
  • (0, 20) - bottom left
  • (10, 10) - center
