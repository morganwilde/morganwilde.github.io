Automation

With the grid from the previous section, we can visualise a collection of points, and we can activate them to change their color. But if you wanted to draw lines, you would have to activate each point manually, which is tedious work. Ideally, we would only have to provide two points through which the line passes. The rest would be taken care of automatically.

For that we need a rule for selecting all the points inbetween the 2 given - a linear function.

y = m*x + k

This function has 2 parameters:

  • m - slope
  • k - vertical coordinate intercept

Since we want to draw lines that include some point a and point b, we can find the 2 parameters as follows.

a = (x1, y1)
b = (x2, y2)
deltaX = x2 - x1
deltaY = y2 - y1

m = deltaY / deltaX
k = y1 - m * x1
